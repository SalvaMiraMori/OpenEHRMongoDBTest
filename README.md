# OpenEHR MongoDB Test

## Descripció
API creada per a fer les següents proves:
- Crear i recuperar composicions d'OpenEHR.
- Crear i recuperar templates d'OpenEHR.
- Obtenir els paths del canonical JSON d'una composició d'OpenEHR.
- Obtenir els paths i els seus corresponents valors del canonical JSON d'una composició d'OpenEHR.



## Configuració
Configura l'arxiu `application.properties`:

```application.properties
spring.application.name=mongodbtest
spring.data.mongodb.uri=[Insertar URI del cluster de MongoDB que es vulgui fer servir]
spring.data.mongodb.database=ehr
server.port=8081
```
